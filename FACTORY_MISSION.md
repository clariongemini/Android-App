## Outcome validation

Portfolio kayıt tutar. **Outcome** ürün performansını kanıtlar:

```
App → users → retention → revenue → cost → ROI
```

Bu veri Play Store / AID / analytics'ten gelir — fabrika şablonunda örnek uygulama adı taşınmaz.

**Sıra:** Repo freeze → hedef projede bootstrap → AID ACTIVE → release → beta → kullanıcı → gelir → outcomes → 30–60 gün veri → repo yeniden değerlendirme.
